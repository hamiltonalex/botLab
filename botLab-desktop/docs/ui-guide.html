Панель управления

✓ Работает
Что это
Выбор параметров расчёта.
Инструмент
APT / ETH / BTC (или ETH-Arb / BTC-Arb / ETH-Avax для одной ноги).
Стратегия
Дельта-нейтральная · 2 ноги (GMX × Hyperliquid) или Одна нога · GMX carry.
Конфигурация
A = short GMX + long HL; B = long GMX + short HL. Выбирается автоматически по большему среднему net APR.
Период / Режим
Период истории 30 / 90 / 365 дней; режим P&L Брутто или Нетто (после издержек).
Капитал
Ввод суммы вручную.
Статус
Все переключатели работают и пересчитывают весь экран.

Доходность и матрица «капитал × плечо»

✓ Работает
Что это
Главный результат стратегии за выбранный период.

Чистый P&L, доходность к капиталу, APR (годовой) и медианная (устойчивая к выбросам), ноционал, $/час, макс. просадка.

Статус

Форвардный бумажный тест (с t0)

✓ Работает
Что это
Бумажная позиция «с текущего момента» — ключевая функция фазы 1.
Для чего
Кнопка «Открыть бумажную позицию» фиксирует t0, инструмент, конфигурацию, капитал и плечо. Далее начисляется реальный P&L: GMX фандинг+заимствование посекундно, HL фандинг в момент почасового расчёта — точно как у реальной позиции. Рисуется кривае equity с t0.
Статус
Работает; позиция сохраняется между запусками приложения (форвард-тест продолжается).

Подпись и разъяснения

Funding-Arb Paper Simulator, версия 0.1.0 — Фаза 1 (бумажный режим). Приложение считает дельта-нейтральную funding-арбитражную стратегию GMX V2 × Hyperliquid на живых биржевых данных и ведёт форвардный бумажный тест с момента t0.

Важно. Фаза 1 не работает с реальными деньгами: нет ордеров, приватных ключей, кастоди. Используются только публичные read-only эндпоинты бирж. Все результаты — бумажные (paper). Числа на иллюстрациях — пример; в приложении данные обновляются с бирж (задержка ≤ 15 минут).

Ограничения этой версии: цена — контекст (межбиржевой базис ≈ 0); комиссии — консервативные значения из аудита; скринер — набор APT/ETH/BTC. Эти пункты дорабатываются в этапах P2–P3.
